# ⚛️ React TS Boilerplate

## 📦 Scripts de instalação e execução

```bash
# Instalar as dependências
yarn install
```

```bash
# Executar aplicação em modo desenvolvimento (localhost:3000)
yarn start
```

```bash
# Gerar arquivos para produção
yarn build
```

```bash
yarn storybook
```

```bash
yarn build-storybook
```
